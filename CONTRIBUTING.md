# Contributing

Thanks for helping improve this project.

## Before you submit

- Keep code comments short and clear.
- Do not commit secrets such as Wi-Fi passwords or MQTT credentials.
- Test the sketch on an ESP32 board when possible.
- Keep changes focused on one topic per pull request.

## Suggested workflow

1. Fork the repository.
2. Create a feature branch.
3. Make your changes.
4. Verify the sketch still compiles.
5. Open a pull request with a short description and screenshots if relevant.

## Reporting issues

When reporting a problem, please include:

- ESP32 board type
- OLED type and size
- Arduino IDE or CLI version
- What you expected to happen
- What actually happened

## Security

If you find a secret or credential in the repository, please report it privately and avoid posting it in a public issue.
